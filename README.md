# Notepad Application (Java)

This project is a simple Notepad application developed using Java Swing.
The program allows users to create, open, edit, and save text files through a graphical user interface. It also supports multiple tabs so users can work on multiple documents at the same time.

## Features

* Create new text files
* Open existing text files
* Save files
* Cut, Copy, and Paste text
* Multiple tabs for editing different files
* Simple and user-friendly interface
* Popup menu support

## Technologies Used

* Java
* Java Swing
* GUI Programming
* File Handling
